# 🎯 AVS 품질 검수 체크리스트

> AVS는 멘토스의 핵심 무기 — 단원별로 직접 돌려보며 **합격/재생성**을 표시하세요.
> **테스트 경로**: `/homework`(숙제 AVS) 또는 `/class/math`(수업 중 AVS 실행)에서 해당 단원 문제를 풀고 해설(AVS) 재생.
> **PCBSA 기준**: P(문제 핵심)·C(접근·조건)·B(핵심 공식 KaTeX 렌더)·S(풀이 흐름·해설)·A(최종 정답 정확).
> 표기: 정답 = avs_answers 커버리지, 강의 = 프리미엄 AI 노트 유무, 🔷 = 도형/그래프 렌더 필수 단원.

## 검수 항목 (각 AVS마다 확인)
1. **PCBSA 구조** — P/C/B/S/A 5단계가 빠짐없이 제대로 들어있는가
2. **정합성** — 해설이 그 문제에 맞는가 (문제와 동떨어진 해설 ✗)
3. **성의** — 풀이가 충실한가 (한 줄짜리·성의없는 해설 ✗)
4. **🔷 도형/함수 렌더** — 기하·함수 단원에서 그래프·도형이 실제로 그려지는가 (안 그려지면 보강)
5. **정답(A)** — 최종 정답이 정확한가

**판정**: 위 항목 모두 양호 → `☐ 합격` / 하나라도 부실·오류·미렌더 → `☐ 재생성` 체크 + 메모(무엇을 고칠지).
